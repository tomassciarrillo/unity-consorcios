# 🏢 Unity - Plataforma de Gestión de Consorcios

Unity es una aplicación web Full Stack diseñada para digitalizar y agilizar la comunicación y administración de edificios de departamentos. Permite la interacción fluida entre administradores y residentes.

## 🚀 Funcionalidades Principales

### 👤 Rol Residente
*   **Cartelera Informativa:** Visualización de comunicados oficiales de la administración con diseño moderno y dinámico.
*   **Mis Expensas:** Consulta de liquidaciones y estado de pago de expensas del departamento.
*   **Mis Reclamos:** Creación y seguimiento de incidentes (ej: filtraciones, luminarias) con actualización de estado en tiempo real.
*   **Reservas de Amenities:** Gestión de reservas de espacios comunes (SUM, Piscina, Parrilla) por día completo, con control de solapamiento de fechas.

### 🔑 Rol Administrador
*   **Residentes Pendientes:** Aprobación y registro de nuevos usuarios en el sistema.
*   **Generar Expensas:** Liquidación y distribución global de expensas del edificio.
*   **Reclamos del Edificio:** Panel de control de incidentes para cambiar estados y gestionar resoluciones.
*   **Agenda de Reservas:** Cronograma integral de ocupación de amenities por unidad funcional.
*   **Cartelera de Anuncios:** Publicación e historial de comunicados oficiales para los vecinos.

## 🛠️ Stack Tecnológico

*   **Frontend:** React (Vite), Tailwind CSS, Lucide Icons, Axios.
*   **Backend:** Spring Boot (Java), Spring Security, JWT (JSON Web Tokens), JPA / Hibernate.
*   **Base de Datos:** PostgreSQL.
